`api.Player` — управление воспроизведением: скорость, громкость, позиция, переключение треков, лайки.

## Методы

| Метод | Описание |
|-------|----------|
| `setSpeed(value)` | Скорость воспроизведения (например, `1.5`) |
| `setVolume(value)` | Громкость от `0` до `1` |
| `seek(seconds)` | Перемотка на позицию в секундах |
| `next()` | Следующий трек |
| `prev()` | Предыдущий трек |
| `like()` | Лайкнуть текущий трек |
| `play()` | Запустить воспроизведение |
| `pause()` | Поставить на паузу |
| `getCurrentTrack()` | Информация о текущем треке |

## Пример

```js
BYM.register({
  id: 'speed-2x',
  name: '2x Speed',
  version: '1.0.0',
}, (api) => ({
  start() { api.Player.setSpeed(2); },
  stop()  { api.Player.setSpeed(1); },
}));
```

<Tip>
Для слежения за сменой трека подпишись на событие `trackChange` в [`api.Events`](/api/events).
</Tip>
