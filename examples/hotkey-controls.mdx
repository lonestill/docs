---
title: "Горячие клавиши"
description: "Глобальные хоткеи для управления скоростью и треками без фокуса YM"
---

Регистрирует глобальные хоткеи через Electron `globalShortcut`. Работают даже когда окно YM свёрнуто или не в фокусе.

## Что показывает

- `api.Hotkeys.register` — регистрация Electron Accelerator комбинаций
- Чтение пользовательских хоткеев из настроек типа `string`
- Связка хоткея с `api.Player` (скорость, треки)
- Toast-фидбек через `api.UI.toast`

## Код

```js hotkey-controls.js
BYM.register({
  id: 'hotkey-controls',
  name: 'Hotkey Controls',
  version: '1.0.0',
  description: 'Глобальные хоткеи для скорости и переключения треков',
  author: 'you',
  settings: [
    { key: 'speedUp',   type: 'string', label: 'Скорость +0.1', default: 'Alt+Up' },
    { key: 'speedDown', type: 'string', label: 'Скорость -0.1', default: 'Alt+Down' },
    { key: 'reset',     type: 'string', label: 'Сброс скорости', default: 'Alt+R' },
    { key: 'next',      type: 'string', label: 'Следующий трек', default: 'MediaNextTrack' },
    { key: 'prev',      type: 'string', label: 'Предыдущий трек', default: 'MediaPreviousTrack' },
  ],
}, (api, getSetting) => ({
  start() {
    api.Hotkeys.register(getSetting('speedUp') ?? 'Alt+Up', () => {
      const v = Math.min(4, parseFloat((api.Player.getSpeed() + 0.1).toFixed(2)));
      api.Player.setSpeed(v);
      api.UI.toast(v.toFixed(2) + '×', 800);
    });

    api.Hotkeys.register(getSetting('speedDown') ?? 'Alt+Down', () => {
      const v = Math.max(0.25, parseFloat((api.Player.getSpeed() - 0.1).toFixed(2)));
      api.Player.setSpeed(v);
      api.UI.toast(v.toFixed(2) + '×', 800);
    });

    api.Hotkeys.register(getSetting('reset') ?? 'Alt+R', () => {
      api.Player.resetSpeed();
      api.UI.toast('1.00×', 800);
    });

    api.Hotkeys.register(getSetting('next') ?? 'MediaNextTrack', () => {
      api.Player.next();
    });

    api.Hotkeys.register(getSetting('prev') ?? 'MediaPreviousTrack', () => {
      api.Player.prev();
    });
  },
  stop() {},
}));
```

## Как использовать

1. Сохрани код в `hotkey-controls.js` и включи плагин.
2. В настройках задай удобные комбинации в формате [Electron Accelerator](/api/hotkeys#формат-combo).
3. Хоткеи будут работать даже когда YM в фоне.

<Warning>
Один `combo` регистрируется один раз глобально. Если другое приложение уже захватило хоткей — регистрация молча провалится. Снимать хоткеи вручную не нужно — `api.Hotkeys.register` отменяется автоматически в `stop()`.
</Warning>
